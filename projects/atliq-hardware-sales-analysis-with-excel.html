AtliQ Hardware Sales Analysis with Excel

Created on July 16, 2025

MS Excel • Power Query • Power Pivot

Project Description

This project features a set of interactive dashboards that provide financial and sales insights for a hardware retail company. It includes monthly and yearly P&L reports, customer performance analysis, and region-wise profitability breakdowns.

These reports help track revenue trends, identify top clients, compare market performance, and support strategic decision-making across geographies and fiscal years.

Project Goals

  • Analyze monthly and yearly P&L reports
  • Evaluate customer performance and profitability
  • Compare market performance across regions
  • Support strategic decision-making with data insights
